# Address Formatter (Python)

Small Python script that asks for street, postal code, and city, then prints a formatted address line.

## File
- `parameterized_3.py`

## Run
```bash
python parameterized_3.py
```

## Example
```text
Street: Main
Postal Code: 10001
City: New York
Your address is: Main St., New York 10001
```
